This page provides an explanation of HTML editors and discusses the features that make them different to a normal text editor. There's also an overview of WYSIWYG editors, which are included in some of the better HTML editors. So anyway, what is an HTML Editor? An HTML editor is a specialized piece of software that assists in the creation of HTML code. Similar to text editors such as Notepad and TextEdit, HTML editors allow users to enter raw text. Most (if not all) professional web developers use an HTML editor to create and maintain their websites. This is because a good HTML editor can boost productivity enormously.
